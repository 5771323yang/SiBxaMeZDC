# 项目概述 python793

本项目是基于Python开发的餐厅点餐预定可视化系统，使用MySQL作为数据库进行数据存储，以Django框架为基础进行开发。

## 技术栈

- **后端**: Python
- **框架**: Django
- **数据库**: MySQL

## 功能模块

- 用户注册与登录
- 菜品浏览与搜索
- 菜品点餐与预定
- 订单管理
- 数据可视化展示

## 系统角色

- 普通用户：浏览、搜索、点餐、预定
- 管理员：菜品管理、订单管理、数据统计

## 运行环境

- Python 3.8 或以上版本
- MySQL 5.7 或以上版本
- 操作系统：兼容Linux和Windows

## 部署步骤

1. 安装Python环境
2. 安装MySQL数据库，并进行配置
3. 克隆项目代码到本地
4. 安装依赖库（pip install -r requirements.txt）
5. 迁移数据库（python manage.py makemigrations; python manage.py migrate）
6. 运行开发服务器（python manage.py runserver）
